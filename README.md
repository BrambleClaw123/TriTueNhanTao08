## Bài tập về nhà các buổi

Mỗi folder là bài tập về nhà của từng buổi học.

---
## Bài tập về nhà buổi 5 - BFS
Cách chạy:
1. Lấy source code về và mở file cần chạy, tìm đến hàm main để sửa initial là trạng thái lúc đầu của trò chơi cần được giải, sau đó bấm Run All.
2. Sau khi chạy, cửa sổ chương trình sẽ hiện lên, bấm "GIẢI" để thuật toán bắt đầu phân tích, sau đó cửa sổ sẽ hiển thị từng step giải bài toán, đồng thời là các hành động để được trạng thái trên màn hình.
3. Sau khi chạy xong, bấm "X" để thoát chương trình.

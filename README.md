## Các thuật toán tìm kiếm (BFS, DFS, IDS) 
### Cách chạy:
1. Lấy source code về và mở file cần chạy, tìm đến hàm main để sửa initial là trạng thái lúc đầu của trò chơi cần được giải, sau đó bấm Run All.
2. Sau khi chạy, cửa sổ chương trình sẽ hiện lên, chọn thuật toán để giải trò chơi, bấm "PLAY" để thuật toán bắt đầu phân tích, sau đó cửa sổ sẽ hiển thị từng step giải bài toán, đồng thời là các hành động để được trạng thái trên màn hình.
3. Sau khi chạy xong, bấm "X" để thoát chương trình.
### Lưu ý:
Sau khi bấm nút "PLAY", tùy thuộc vào thuật toán đã lựa chọn mà chương trình sẽ có thời gian chạy khác nhau (có thể nhanh hoặc chậm).
